# Petra examples

These examples follow the same idea as Chi's `_examples`: each folder shows one
thing instead of one large application trying to show everything.

## Apps

- `mvcweb`: Chi, controllers, Petra layouts/components, dev reload, embedded production mode.
- `tailwind`: Tailwind compiled with Vite, served by Petra, and refreshed through Petra's stylesheet-aware dev reload.
- `alpine`: Alpine bundled with Vite, served by Petra, and page-reloaded on JavaScript changes in dev mode.
- `forms`: server-rendered GET/POST form flow with validation and re-rendered errors.
- `debugerrors`: rendered Petra debug pages for broken page and component templates.
- `htmx-todo`: HTMX partial swaps for a tiny todo app using `go-htmx`.

Each example has its own README and can be run from its folder.
