About

What this example is meant to show.

Petra can sit inside a normal Go web app without a frontend framework. Use embedded assets in production, disk-backed templates in development, and the same controllers in both modes.

Development path

Run with go run . -dev. Template edits send reload; static edits send reload_assets.

Production path

Run without -dev. Templates and static assets come from embed.FS.
