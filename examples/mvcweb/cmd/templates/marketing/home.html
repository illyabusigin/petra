Petra example

A small MVC web app using Petra.

This example keeps the application shape familiar: a command starts the server, controllers own routes, and templates live in a nested tree with layouts and components.

Model

Page data is plain Go structs. The controller builds the context passed to Petra.

View

Templates use one root layout, shared components, and page-specific content blocks.

Controller

Each controller owns its routes and calls one shared render helper.
